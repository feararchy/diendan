# 💻 Kingmate Tech Forum
Dự án diễn đàn công nghệ mã nguồn mở được xây dựng bằng Python/Django.

## ✨ Tính năng chính
- 🔐 **Hệ thống tài khoản:** Đăng ký, Đăng nhập, Phân quyền Admin/Thành viên.
- 📝 **Bài viết:** Trình soạn thảo CKEditor, Upload ảnh, Thu nhỏ ảnh (Thumbnail).
- 💬 **Tương tác:** Bình luận, Thả tim (Like) ngay trang chủ.
- 📁 **Chuyên mục:** Lọc bài viết theo danh mục thông minh.
- 👥 **Thành viên:** Danh sách thành viên và Bảng vinh danh số lượng bài đăng.

## 🛠️ Công nghệ sử dụng
- **Ngôn ngữ:** Python 3.x
- **Framework:** Django 6.x
- **Cơ sở dữ liệu:** PostgreSQL (Mã nguồn mở)
- **Giao diện:** Bootstrap 5, Javascript

## 🚀 Hướng dẫn cài đặt
1. Clone dự án: `git clone https://github.com/feararchy/diendan.git`
2. Cài đặt thư viện: `pip install -r requirements.txt`
3. Cấu hình database PostgreSQL trong `settings.py`
4. Chạy lệnh: `python manage.py migrate`
5. Khởi chạy: `python manage.py runserver`
